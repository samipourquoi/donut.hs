A bad recreation of [Andy Sloane's donut.c](https://www.youtube.com/watch?v=DEqXNfs_HhY) in Haskell.

## Compile
You will need the [Haskell Platform](https://www.haskell.org/platform/) to compile the code.

```shell-script
$ cabal build
$ cabal exec donut-hs
```
